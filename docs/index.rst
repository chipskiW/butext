================
Welcome to **BUtext**!
================

**BUtext** focuses on text processing techniques commonly used in Natural Language Processing (NLP).  
This documentation will cover the following topics:

* Tokenization  
* Relative Frequency  
* Term Frequency–Inverse Document Frequency (TF–IDF)
* Logistic Regression
* SVM
* PCA
